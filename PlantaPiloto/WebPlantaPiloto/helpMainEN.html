Main page field

  1. Connection String for the data base connection.
  2. Link to all tables of the project database.
  3. Project variables in real time. The controls allow you to indicate which ones will be shown in the graph on the right (10).
  4. Link to the "About" information.
  5. Button to display the help page.
  6. Field for changing the variables. In the drop-down menu the variable to be edited will be selected and in the right text the new value that it will take.
  7. Amount of data to display in the graphs and number of graphs to display.
  8. Language change menu.
  9. Visual theme change menu.
  10. Section where the main graph is shown with the variables marked in the table on the left.
